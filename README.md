# FuckDalamudCN


解除国服 Dalamud 中不合理的数据上传、插件封锁。还额外提供插件管理器 Github 第三方库加载和下载代理/加速服务

## 使用方法

请前往此 Repo 获取仓库地址 [https://github.com/decorwdyun/DalamudPlugins](https://github.com/decorwdyun/DalamudPlugins)
